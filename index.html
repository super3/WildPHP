a PHP based IRC Bot for games, logging, and management

Free and Open Source. Download .ZIP More Info

PHP Based

No complicated or ugly scripting langage that other bots use. Use a modern language to edit or extend it for your use.

Object Orientated

The bot is fully object orientated. So it's really easy to extend using a structure that everyone uses.

Easy to use

For everyone, even those who don't know anything about programming or PHP can use the bot via the nice web gui.

Cross-Platform

No platform depending features. The bot will run on any OS where PHP is installed. Will run in a Browser or a Command Line.

Extendable

Easily add new command, command packages, and bots and use them with a clicks.

Collaborative

We are friendly. Just get in touch with us, or take a look at the source on github and help us improve the bot.
